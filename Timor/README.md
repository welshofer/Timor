# Timor

A macOS application for managing Spotify playlists with support for adding, deleting, reordering tracks, and managing playlists.

## ⚠️ CRITICAL WARNING ⚠️

**THIS APPLICATION CAN PERMANENTLY DELETE YOUR SPOTIFY PLAYLISTS AND TRACKS**

This software has the ability to:
- **DELETE entire playlists from your Spotify account**
- **Remove tracks from your playlists**
- **Modify your Spotify library in ways that CANNOT be undone**

**USE AT YOUR OWN RISK.** The author(s) assume NO responsibility for:
- Lost or deleted playlists
- Removed tracks
- Any unintended changes to your Spotify library
- Any data loss whatsoever

**THERE IS NO UNDO FUNCTIONALITY.** Actions taken by this app directly modify your Spotify account and cannot be reversed.

## Features

- View and browse Spotify playlists
- Add tracks to playlists
- Like/unlike tracks
- Delete tracks from playlists
- Reorder playlist items
- Delete entire playlists

## Requirements

- macOS 26 or later
- Spotify Premium account (some features may require Premium)
- Spotify Developer App credentials
